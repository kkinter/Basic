### What is refactoring ?
- Restructure code
	- Easier to read
	- More efficient
	- Less duplication
- Improve perfoemance
- Does the same thing

### TDD Refactoring
- TDD makes refactoring easy
- Run tests to ensure code still works

### Areas of refactoring
- `TagViewSet` / `IngredientViewSet`
	- Very similar code
	- Refactor using inheritance
